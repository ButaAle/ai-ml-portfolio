# AI/ML Portfolio

This repository documents my AI/ML career improvement plan using GitHub portfolio projects, Kaggle, DrivenData, Zindi, and open-source tools.

The goal is to create public CV evidence through reproducible code, notebooks, validation metrics, documentation, error analysis, and portfolio-ready project summaries.



## Portfolio Strategy

My goal is not only to train models, but to show professional machine learning workflow:

- Clear problem statement
- Reproducible code
- Clean data analysis
- Baseline model
- Validation strategy
- Metric interpretation
- Error analysis
- README documentation
- Honest limitations and next steps

## Platforms

| Platform | Why I use it | Portfolio evidence |
|---|---|---|
| Kaggle | Public notebooks, datasets, competitions, leaderboard practice | Notebook, metric, GitHub mirror |
| DrivenData | Practical data science competitions with real-world/social-impact problems | Clean solution report and validation strategy |
